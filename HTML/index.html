CMYK is a color model, used in printing. It stands for cyan, magenta, yellow and key, which is black. RGB is a color model where red, green and blue are used to create wide array of colors. It's main purpose is displaying colors on electronic devices, such as computers and phones. In grayscale imaging each pixel represents only an amount of light, as name suggests it is composed of shades of gray, it is used to show brightness, not color. 2. RAW image file is minimally processed data from a image sensor, RAW files are used to give more control to the photographer, they have more shades of colors, higher image quality. 3. RGB is used in web and applications. If image is in wrong format for web or apps they can be converted 4. Vector images can be added through img element or using svg element 5. Small file size, highly scalable, resolution independence. 6. Lazy loading is delaying loading images or objects till they are needed, this improves performance and resources.
